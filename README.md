# Blackjack Game

## About the Project:
Our class was tasked with creating a working blackjack game.  I chose to incorporate various elements such as a hidden dealer card and the use of multiple decks, to make it more realistic.

## Built With:
- HTML
- CSS / Bootstrap
- Javascript
